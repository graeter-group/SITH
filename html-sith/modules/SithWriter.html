SITH.SithWriter.calculate_error(sith: SITH.SITH.SITH) → Tuple

Performs error calculation

Error calculation is based off of the discrepancy between the energy given by the input geometry optimization calculations and the total stress energy calculated by SITH analysis using the Hessian and Δq.

Parameters

sith (SITH) – analyzed SITH object

Returns

(expected stress energy, stress energy error, %Error)

Expected Stress Energy: Total E deformed structure from input .fchk - total E reference structure from input .fchk

Stress Energy Error: calculated stress energy - Expected Stress Energy

%Error: Stress Energy Error / Expected Stress Energy

Return type

Tuple

SITH.SithWriter.write_all(sith: SITH.SITH.SITH, filePrefix='') → bool

Write all SITH output files: summary, energies, error, and Δq in the parent directory of the reference structure.

This function simply calls all SithWriter.write methods which take a SITH object as input. Units are Angstroms, degrees, and Hartrees.

Parameters

  • sith (SITH) – analyzed SITH object

  • filePrefix (str, optional) – prefix for file output. Defaults to ‘’.

Returns

If all files were successfully written.

Return type

bool
